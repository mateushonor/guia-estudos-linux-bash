# Shell Bash

O shell é um programa que permite a comunicação entre o usuário e o sistema operacional.

No Linux, o shell mais comum é o **Bash (Bourne Again Shell)**.

Ele interpreta comandos digitados no terminal e executa ações no sistema.

---

# Exemplo de uso

Quando digitamos um comando como:

```bash 
ls

````markdown
O Bash executa o comando `ls`, que significa **list** (listar).

Ele mostra todos os arquivos e diretórios presentes no diretório atual.

```bash
Documentos  Downloads  Imagens  projetos
