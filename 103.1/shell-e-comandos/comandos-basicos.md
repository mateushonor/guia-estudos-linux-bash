# Comandos básicos do Bash

Este arquivo apresenta alguns comandos básicos usados no terminal Linux.

---

# Comando ls

Lista arquivos e diretórios.

```bash
ls
```

Exemplo de saída:

```bash
Documentos Downloads Imagens projetos
```

---

# Comando pwd

Mostra o diretório atual.

```bash
pwd
```

Saída:

```bash
/home/mateus
```

---

# Comando whoami

Mostra o usuário atual.

```bash
whoami
```

Saída:

```bash
name
```

---

# Comando date

Mostra a data e hora do sistema.

```bash
date
```

Exemplo de saída:

```bash
Mon Mar 24 14:32:10 -03 2026
```
